# A.ENG - تطبيق إدارة الديون Android

نسخة جديدة بألوان مختلفة بالكامل، واسم وأيقونة جديدة.

## التعديلات
- اسم التطبيق: **A.ENG**
- أيقونة جديدة باسم A.ENG
- ألوان جديدة بالكامل: تركواز + برتقالي + خلفية بيج فاتحة
- إجبار كل النصوص أن تكون داكنة وواضحة
- تعطيل الوضع الليلي التلقائي في Android WebView
- PIN عند فتح التطبيق مباشرة
- الرمز الافتراضي: `5135`

## بناء APK من GitHub
1. ارفع الملفات إلى المستودع.
2. افتح تبويب Actions.
3. شغل Build Daftari Pro APK.
4. بعد الانتهاء حمّل Artifact.
5. ثبّت ملف app-debug.apk على الهاتف.
